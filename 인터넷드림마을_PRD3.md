## 2. 미로 인쇄 워크시트 PDF

> **설계 배경:** 스마트폰 중독 예방 프로그램에서 웹/모바일 게임 제공은 모순적이라는 현장 피드백 반영.
> 미로 활동은 출력물로 제공 — 화면 없이 종이와 펜으로 수행.

### 2.1 기존 자산 활용

**원본 프로젝트:** `브레인-리셋_-10단계-미로-프로젝트` (Vite + React + TypeScript)

**활용 파일:**
- `src/data/levels.ts` — 10단계 미로 데이터 (벽 좌표, 그리드 크기)
- `src/components/Worksheet.tsx` — 인쇄용 SVG 렌더링 로직 참고

**미사용:**
- `Maze.tsx` (웹 인터랙티브 게임 컴포넌트 — 이번 방향과 불일치)
- Vite 설정 파일 일체

### 2.3 PDF 산출물 구성

**2종 PDF 제공:**

| 종류 | 대상 | 정답 경로 포함 |
|---|---|---|
| 학생용 | 학생 직접 출력 또는 교사 배포 | 없음 |
| 교사용 | 교사 전용 (정답 확인·채점용) | 있음 (빨간 점선) |

**각 PDF 내 페이지 구성 (레벨당 1페이지):**
```
헤더: "인터넷 드림마을 — 뇌 훈련 미로 Lv.{N}: {테마명}"
본문: 미로 SVG (A4 기준 최대 크기)
하단: 학생 이름 기입란 / 날짜 기입란
      (교사용만) 정답 경로 빨간 점선 오버레이
```

**파일명:**
```
미로_학생용_10단계.pdf
미로_교사용_10단계_정답포함.pdf
```

## 3. 꽃 감정 SVG 고도화

### 3.1 현재 (Phase 1) vs Phase 3

| | Phase 1 | Phase 3 |
|---|---|---|
| 방식 | 프리셋 8종 고정 | 프리셋 베이스 + AI 파라미터 조절 |
| 다양성 | 8가지 | 무한 변형 |
| 품질 | 완벽 일관성 | 일관성 유지 + 다양성 |

### 3.2 AI 파라미터 조절 방식

**Gemini 입력:** 일기 텍스트 분석 결과 (감정 키워드 + valence 수치)

**Gemini 출력 (JSON):**
```json
{
  "flower_type": "sunflower",
  "bloom_degree": 0.85,
  "primary_color": "#FFD700",
  "secondary_color": "#FFA500",
  "petal_count": 16,
  "stem_lean": -5,
  "leaf_size": 1.2,
  "droop": 0.0
}
```

**SVG 파라미터 설명:**

| 파라미터 | 범위 | 의미 |
|---|---|---|
| bloom_degree | 0.0~1.0 | 개화도 (0=봉오리, 1=만개) |
| primary_color | hex | 꽃잎 주색 |
| secondary_color | hex | 꽃잎 보조색/그라데이션 |
| petal_count | 5~20 | 꽃잎 수 |
| stem_lean | -15~15 | 줄기 기울기 (도) |
| leaf_size | 0.5~1.5 | 잎 크기 배수 |
| droop | 0.0~1.0 | 고개 숙임 정도 |

**감정 → 파라미터 매핑 예시:**

| 감정 | bloom | droop | stem_lean |
|---|---|---|---|
| 기쁨 | 1.0 | 0.0 | 0 |
| 슬픔 | 0.3 | 0.8 | -10 |
| 불안 | 0.2 | 0.2 | 5 |
| 무기력 | 0.1 | 1.0 | -15 |
| 평온 | 0.7 | 0.0 | 0 |

## 4. 스마트폰 사용량 × 정서 상관관계 그래프

### 4.1 데이터 수집

**매일 학생 대시보드 접속 시:**
```
"오늘 스마트폰 몇 시간 썼어?"
[슬라이더: 0~12시간, 0.5시간 단위]
[건너뛰기] 버튼 제공 (강제 아님)
```

**수집 데이터:**
```sql
smartphone_usage
  - id
  - user_id
  - usage_hours    -- float (0~12, 0.5 단위)
  - recorded_date  -- date
  - created_at
```

### 4.2 그래프 구성

**형태:** 상하 이중 선 그래프 (Recharts), 날짜 축 동기화

```
[위 그래프] 스마트폰 사용 시간 (시간/일)
            선 색상: 주황 (#FF8A65)
            Y축: 0~12시간

[아래 그래프] 정서 안정도 (자가진단 점수 또는 일기 감정 valence)
             선 색상: 파랑 (#90CAF9)
             Y축: 0~100점

X축: 날짜 (최근 30일 기본, 전체 조회 가능)
```

**메타인지 문구:**
```
"폰을 덜 쓴 날, 기분이 어땠는지 보이나요? 📊"
```

**표시 위치:**
- 학생 대시보드 하단 섹션
- 관리자 학생 상세 페이지

## 5. 수퍼바이저 대시보드

### 5.1 기능 목록

#### 관리자 계정 관리
```
목록 뷰:
  - 기관명 / 담당자명
  - 가입일
  - 마지막 로그인일
  - 구독 만료일 (D-30 이하: 주황, D-7 이하: 빨강)
  - 산하 학생 수
  - 상태 (활성 / 만료 / 정지)

개별 클릭 → 상세 페이지:
  - 위 정보 상세
  - 구독 만료일 수동 연장 (날짜 선택 → 저장)
  - 관리자 비밀번호 초기화 (임시 비밀번호 발급)
  - 계정 정지 / 활성화 토글
```

#### 자동 알림
```
만료 D-30: 수퍼바이저 이메일 알림
만료 D-7:  수퍼바이저 이메일 알림 (+ 해당 관리자에게도 알림)
```

#### 전체 서비스 통계
```
KPI 카드:
  - 전체 활성 기관 수
  - 전체 학생 수
  - 이번 달 신규 가입 기관

차트:
  - 월별 활성 기관 수 추이 (막대 그래프)
  - 월별 학생 수 추이 (선 그래프)
```

### 5.2 데이터 모델

```sql
-- admins 테이블 기존 컬럼 활용
admins
  - id
  - email
  - institution_name   -- 기관명
  - contact_name       -- 담당자명
  - subscription_expires_at
  - status             -- active | suspended | expired
  - created_at
  - last_login_at
```

## 6. PDF 저장 — Phase 3 추가 범위

### 6.1 관리자 학생 개별 리포트 PDF

**버튼 위치:** 관리자 → 학생 상세 페이지 상단 "PDF 출력" 버튼

**PDF 포함 내용 (Phase 3 완성 기준 풀스펙):**
- 헤더: "인터넷 드림마을" + 기관명
- 학생 기본정보 (이름, 참여 기간, 담당 관리자)
- 자가진단 점수 추이 그래프 (Recharts → html2canvas 캡처)
- 마음일기 감정 키워드 월별 요약
- CIREF 세션 요약 (최근 3회 이미지 + 리플렉션 Q1)
- 위기신호 이력 (감지 일시, 키워드, 처리 여부)
- 스마트폰 사용량 × 정서 상관관계 그래프
- 미로 워크시트 다운로드 횟수 (해당 기관 기준)
- 푸터: 출력 날짜 + 페이지 번호

**파일명:** `{기관명}_{학생명}_리포트_{YYYY-MM-DD}.pdf`

### 6.2 전체 학생 요약 리포트 PDF

**버튼 위치:** 관리자 대시보드 상단 "전체 요약 PDF" 버튼

**PDF 포함 내용:**
- 전체 학생 수 / 활성 학생 수 / 평균 자가진단 개선율 KPI 카드
- 학생별 1줄 요약 테이블 (이름, 자가진단 수준, 마지막 접속일, 위기신호 여부)

**파일명:** `{기관명}_전체요약_{YYYY-MM-DD}.pdf`

### 6.3 위기신호 이력 리포트 PDF

**버튼 위치:** 관리자 대시보드 → 위기신호 탭 상단

**PDF 포함 내용:**
- 기간 내 위기신호 전체 목록 (학생명, 감지 일시, 키워드, 처리 여부)

**파일명:** `{기관명}_위기신호이력_{YYYY-MM-DD}.pdf`

## 7. 개발 순서

1. **미로 PDF 생성** — `levels.ts` 이식 + SVG 렌더링 → jsPDF/pdfmake로 학생용·교사용 PDF 출력
2. **관리자 다운로드 UI** — 학습 자료 섹션 + 학생용/교사용 버튼
3. **꽃 SVG 파라미터화** — 기존 프리셋 SVG 파라미터 분리 + Gemini 연동
4. **스마트폰 사용량 수집** — 슬라이더 UI + DB 저장
5. **상관관계 그래프** — Recharts 이중 선 그래프
6. **수퍼바이저 대시보드** — 관리자 목록, 구독 연장, 비밀번호 초기화, 통계
7. **관리자 학생 개별 리포트 PDF** — `usePDFExport` 훅 + 풀스펙 printRef 구성
8. **전체 학생 요약 리포트 PDF**
9. **위기신호 이력 리포트 PDF**
