왜 인터넷 드림마을인가

청소년 스마트폰 과몰입,
해결책이 필요합니다

추상적인 경고가 아니라, 뇌 과학 기반의 구체적 미래 시각화로 행동 변화를 이끌어냅니다.

🧠

도파민 보상 회로 재공고화

EFT 기법으로 먼 미래 보상을 현재처럼 체감하게 유도. 지연 할인(Delay Discounting)을 극복합니다.

🎯

CIREF 프레임워크

단서 유발 회상과 일화적 미래 사고를 결합한 4단계 대화로 진정한 행동 변화 동기를 만듭니다.

🌱

정서 기록 + AI 시각화

매일의 감정을 꽃으로 표현하고, 갤러리로 쌓아가며 자신의 감정 변화를 직접 목격합니다.

👨‍⚕️

전문가와의 협력

AI는 보조 도구. 위기 신호 감지 시 즉시 담당 상담사에게 알림. 최종 판단은 사람이 합니다.

학생 핵심 모듈

4가지 핵심 기능으로
매일 성장합니다

학생 대시보드의 4개 모듈은 감정 기록부터 미래 시각화까지 유기적으로 연결됩니다.

🌸

마음일기

오늘의 감정을 자유롭게 쓰면 AI가 감정을 분석해 꽃 이미지로 시각화합니다. 날짜별로 쌓이는 감정 꽃 갤러리로 나의 감정 변화를 한눈에 봅니다.

AI 감정 분석 꽃 SVG 생성 위기신호 감지 갤러리
🔮

미래의 나

4단계 CIREF 대화로 5년 후, 10년 후의 나를 AI 이미지로 생성합니다. 스마트폰을 내려놓은 미래를 눈으로 보며 행동 변화 동기를 강화합니다.

CIREF 4단계 AI 이미지 생성 리플렉션 노트 변천사 갤러리
🎮

미로 풀기

10단계 뇌 훈련 미로로 스마트폰 대신 건강한 디지털 대안 활동을 제공합니다. 시야 제한 + 타이머로 중학생도 도전하는 난이도를 구현합니다.

10단계 레벨 시야 제한 포인트/뱃지 일일 15분 제한
📊

자가진단

SAS + PHQ-A 기반 10문항 척도로 스마트폰 의존도와 정서 상태를 정기 측정합니다. 스마트폰 사용 시간과 정서 안정도의 상관관계 그래프를 제공합니다.

10문항 척도 추이 그래프 상관관계 분석 수준별 알림
EFT 핵심 모듈

4단계 CIREF로
미래를 그립니다

마법사 UI + AI 재질문으로 청소년이 자신만의 긍정적 미래를 구체적으로 상상하도록 안내합니다.

1

시공간 묘사

스마트폰을 내려놓은 미래의 장소와 상황을 구체적으로 떠올립니다.

"미래의 너는 어디서 누구와 뭘 하고 있어?"
2

감각/감정 예측

그 미래 장면에서 느끼는 감정과 자신에 대한 생각을 탐색합니다.

"그 장면에서 기분이 어때?"
3

정체성 탐색

해결된 미래의 자신이 꿈꾸던 어떤 어른의 모습과 닮았는지 탐색합니다.

"어떤 어른의 모습과 닮았어?"
4

대체 활동

스마트폰 대신 푹 빠져서 하고 싶은 건강한 대안 활동을 찾습니다.

"스마트폰 대신 뭘 해보고 싶어?"
이미지 생성 흐름
1

4단계 완료 → 5년 후 이미지 생성

Gemini API가 CIREF 답변을 분석해 이미지 프롬프트를 자동 생성. 수채화 스타일 1인칭 시점 이미지 출력.

2

리플렉션 노트 — 3문항 순차 표시

"이 미래의 나를 보고 어떤 느낌이 들어?" → "뭔가 바꾸고 싶어?" → "오늘 할 수 있는 한 가지는?"

3

"10년 후의 나도 볼까요?" 선택

학생 선택에 따라 10년 후 이미지 추가 생성. 거절 시 5년 후만 저장. API 비용 절감.

4

슬라이드쇼 갤러리에 누적 저장

첫 세션부터 최근 세션까지 변천사를 슬라이드로 확인. 성장 서사를 직접 체감.

수퍼바이저

시스템 운영자

  • D-30 / D-7 자동 알림
  • 관리자 비밀번호 초기화

관리자

  • 학생 계정 일괄 생성 / CSV 업로드

  • 마음일기 · 자가진단 · 미래이미지 열람
  • 위기신호 즉시 이메일 알림
  • 학생별 메모 기록
  • 게임/CIREF 세션 횟수 설정

4단계 개발 계획

MVP부터 모바일 앱까지 단계적으로 완성합니다.

PHASE 1 — MVP

핵심 기능 완성

  • 3-Tier 계정 시스템 + RLS
  • 마음일기 + AI 감정 분석
  • 감정 꽃 프리셋 8종
  • 자가진단 10문항 척도
  • 관리자 기본 대시보드
  • 위기신호 감지 + 이메일 알림
  • 수퍼바이저 대시보드 기초
PHASE 2 — EFT 핵심

미래 자화상 모듈

  • CIREF 4단계 마법사 UI
  • AI 이미지 생성 (Gemini)
  • 텍스트 앵커 외모 일관성
  • 5년후 → 10년후 프로그레시브 리빌
  • 리플렉션 노트 3문항 순차 표시
  • 슬라이드쇼 갤러리 (변천사)
  • 월 5회 세션 제한 (관리자 조정)
PHASE 3 — 게임 & 고도화

미로 · SVG · 그래프

  • 10단계 미로 게임 이식
  • 시야 제한 + 타이머 난이도 강화
  • 포인트 · 뱃지 5종
  • 꽃 SVG AI 파라미터 고도화
  • 스마트폰 사용량 자기보고
  • 상관관계 이중 선 그래프
  • 수퍼바이저 구독 관리 완성
PHASE 4 — 앱 & 부모

모바일 앱 + 부모 모니터링

🔐

DB 레벨 데이터 격리

Supabase RLS 정책으로 학생은 본인 데이터만 조회 가능. 타 학생 데이터 DB 레벨 완전 차단.

🔒

민감 데이터 암호화

일기 텍스트 등 미성년자 내면 데이터는 암호화 저장. CIREF 답변 원문 보호.

👨‍⚕️

AI는 보조 도구

위기신호 자동 감지는 보조 역할. 최종 판단과 개입은 반드시 전문 상담사가 수행.

👪

부모 열람 제한

부모는 수준/접속/사용시간만 열람. 일기 전문·미래 이미지는 학생 동의 없이 열람 불가.

📋

임상 타당도 보호

SAS·PHQ-A 기반 자가진단 문항 하드코딩. 관리자 임의 수정 불가로 임상 타당도 보장.

🖼

이미지 접근 제한

생성된 미래 이미지는 학생 본인 + 담당 관리자만 열람. Supabase Storage RLS 적용.
